# Memory Index

Arquivos canonicos de memoria deste projeto:

- memory/project.md: memoria principal do projeto (stack, convencoes, arquitetura).
- memory/napkin.md: padroes praticos aprendidos e runbook.
- memory/user.md: fatos estaveis sobre o usuario.
- memory/preferences.md: preferencias operacionais.
- memory/people.md: atores e responsabilidades.
- memory/decisions.md: resumo humano de decisoes.
- memory/decisions.csv: trilha estruturada para revisoes.
