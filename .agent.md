# Security Auditor Agent

Audita vulnerabilidades de seguranca em codigo, arquitetura e infraestrutura.

## Como usar

```
/security revisar autenticacao em services/api/src/auth.ts
/security SQL injection em queries de banco de dados
/security checklist de seguranca para producao
/security vulnerabilidades OWASP em [arquivo ou pasta]
```

## Uso modular explicito (opcional)

```
/security module:auth-idor revisar login e JWT
/security module:db-deps validar RLS e queries SQL
/security module:data-secrets verificar vazamento de credenciais em logs
/security module:all auditoria completa
```
