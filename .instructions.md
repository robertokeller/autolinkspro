# AutoLinks Copilot Instructions

## Skills Disponiveis

### Security Skill (/security)
Use para auditoria de seguranca em autenticacao, autorizacao, banco, APIs, secrets, deploy e infraestrutura.

Ativacao automatica padrao:
- Sempre ativar em tarefas sobre banco de dados, logins, autenticacao/autorizacao, JWT, sessao, secrets, credenciais, APIs, webhooks, upload, deploy, cloud e dependencias.
- Sempre de forma modular (apenas modulos necessarios), com `baseline` obrigatorio.

Modulos:
- `baseline` (sempre)
- `web-client`
- `auth-idor`
- `auth-lifecycle-email`
- `api-injection`
- `upload-malware`
- `webhooks-integrations`
- `data-secrets`
- `db-deps`
- `docker`
- `nginx-deploy`
- `ddos-rate-limit`
- `desktop-extension`
- `incident-response`
- `cloud-infra`
- `appsec-assurance`

Exemplos:
- `/security revisar autenticacao em services/api/src/auth.ts`
- `/security module:db-deps revisar RLS e SQL injection`
- `/security module:data-secrets auditar credenciais e logs`
- `/security module:all checklist completo de producao`

### Scalability Skill
Use para otimizar performance, APIs, banco de dados, cache e infraestrutura.

### PostgreSQL Best Practices Skill
Use para otimizar queries, indexes e schema no Supabase/Postgres.
